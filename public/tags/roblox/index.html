How to Fix Roblox Error Code 277 (Connection Lost)

How to Fix Roblox Error Code 277 Error Code 277 in Roblox is a “General Connection Error.” It means your device cannot communicate with the Roblox game server. It’s infamous for happening on tablets and mobile, but PC users get it too. Why is this happening? Weak Wi-Fi signal. Corrupted cache files on your device. Roblox servers are down. Fix 1: Clear Roblox Cache (PC) Press Win + R and type %localappdata%\Roblox\Logs. Delete all files in the folder. Go back to the Roblox folder and delete the content of Versions. Restart Roblox.
